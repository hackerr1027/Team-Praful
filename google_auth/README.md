# Fleet Management System

A centralized, rule-based digital hub that optimizes the lifecycle of a delivery fleet, monitors driver safety, and tracks financial performance.

## Features

### Current Implementation
- **User Authentication**: Login and registration system
- **Role-based Access**: Fleet Manager, Dispatcher, Safety Officer, and Financial Analyst roles
- **Responsive Design**: Mobile-friendly interface
- **Form Validation**: Client-side validation for all inputs

### User Roles
- **Fleet Manager**: Oversees entire fleet operations, vehicle maintenance, and driver management
- **Dispatcher**: Manages route planning, scheduling, and real-time dispatch operations
- **Safety Officer**: Monitors driver safety, compliance, and incident reporting
- **Financial Analyst**: Tracks financial performance, cost analysis, and budget optimization

### Planned Features
- Fleet vehicle management
- Driver safety monitoring
- Financial performance tracking
- Route optimization
- Maintenance scheduling
- Real-time GPS tracking

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional, for development)

### Installation
1. Clone or download this repository
2. Navigate to the `odoo` folder
3. Open `index.html` in your web browser

### Usage
1. **Register**: Click on the "Register" tab to create a new account
2. **Login**: Use your credentials to access the system
3. **Select Role**: Choose your role during registration (Fleet Manager, Dispatcher, Safety Officer, or Financial Analyst)
